## A note about this project

MERA is a personal project — built out of genuine curiosity and a desire to create something useful. It has no affiliation with any company or organization, and there's no intention, now or ever, to turn it into a commercial product.

The code here is 100% AI-assisted, developed with tools like GitHub Copilot and Claude Code woven into the workflow throughout. This isn't something I'm hiding — quite the opposite. I think AI-assisted development is a legitimate and exciting way to build software, and MERA is a honest reflection of that approach.

If you've stumbled across this project and feel like contributing — whether that's a bug report, a feature idea, a pull request, or just a thought — you're genuinely welcome here. Good software tends to grow from good conversations, and I'd love to hear from you.
